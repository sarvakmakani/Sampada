
# Sampada

Sampada is a lightweight SaaS platform that helps local sellers create online storefronts and receive orders via WhatsApp.

## Features

- Seller authentication
- Store creation
- Product management
- WhatsApp ordering
- QR code sharing
- Analytics dashboard
- Category filtering
- Search functionality

## Tech Stack

Frontend: Next.js, TailwindCSS  
Backend: Node.js, Express, MongoDB  
Hosting: Vercel + Render + MongoDB Atlas

## Setup

1. Clone repo
2. Install dependencies
3. Add .env variables
4. Run frontend and backend
